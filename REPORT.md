## 最致命的三类问题

### 1. 未来函数泄漏（72% 命中率）

```python
# ❌ 错误：未 shift，当天收盘价参与当天决策
df['signal'] = (df['close'] > df['SMA_20']).astype(int)

# ✅ 正确：shift(1) 确保只用历史数据
df['signal'] = (df['close'].shift(1) > df['SMA_20'].shift(1)).astype(int)
```

这不是小问题——**用它回测出来的任何收益数字都是假的。**

### 2. 无交易成本（85% 命中率）

```python
# ❌ 缺失：策略按收盘价成交，不考虑手续费+滑点
pnl = (df['close'].diff() * position).cumsum()

# ✅ 应包含：至少 0.1% 手续费 + 0.05% 滑点
cost = abs(position.diff()) * price * 0.0015
pnl = (df['close'].diff() * position).cumsum() - cost.cumsum()
```

对于高频策略，交易成本可以把 50% 年化收益吃成负数。

### 3. 无止损机制（68% 命中率）

没有一个被扫描的策略在代码中包含完整的止损逻辑。这意味着回测里不会出现"爆仓"——但实盘会。
